  • 렌더링

    [렌더링은 컴퓨터 프로그램을 사용하여 모델(또는 씬(scene))로부터 영상을 만들어내는 과정을 말한다.] 라고 위키피디아에 나와있다. 단순히 html을 렌더링 한다 라고 하면서 거기에만 쓰는 것이 아니라는 이야기이다. 하지만 우리는 html 렌더링에 대해서 알아보겠다.

    html 렌더링은 클라이언트측(웹브라우저)에서 서버측으로 html 문서의 요청을 보내고 그에 대한 응답의 결과로 받은 html 문서를 분석하고 해석하여 화면에 보여주는 일을 한다. 이 과정에서 html 내부의 구조를 파악하기도 하며 css를 분석하여 적용시키는 작업도 한다. 간단하게 요약하면 이렇다.
    브라우저의 렌더링 엔진에는 대표적으로 웹킷(Webkit)과 게코(Gecko)가 있다. 파이어폭스는 게코를, 사파리와 크롬은 웹킷을 사용한다.

    자세한 내용을 알고 싶다면 첨부한 링크를 들어가서 읽어보길 추천한다. 굉장히 자세하게 나와있어서 스압을 받을것이다.
